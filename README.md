# Segunda Infancia

Sitio web estático premium para el equipo de fútbol **Segunda Infancia**.

## Stack

- HTML
- CSS
- JavaScript

## Ejecutar localmente

Solo abre `index.html` en el navegador o usa un servidor estático:

```bash
npx serve .
```
